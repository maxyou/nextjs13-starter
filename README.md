A Next.js-13 starter with the new app router. I'm trying to add more features to make it a complete starter for future projects.

## Tech Stack

- Next.js + App-Router
- Prisma
- PostgreSQL
- Restful-API + SWR
- GraphQL-API + Yoga + URQL

## Done

- Yoga as graphQL server + URQL as graphQL client
- SWR for Restful-API
- Prisma to access PostgreSQL
- todolist CRUD
- authentication + Google Login
